# Changelog

All notable changes to the **FT Blocks** plugin will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Additional custom blocks for photography websites.
- Advanced customization options for existing blocks.
- Performance optimization strategies.

## [1.0.0] - 2026-01-08

### Added
- Initial plugin release.
- **Hero Block**: Customizable content, styling, and server-side rendering.
- **Bilingual support**: Full i18n support for English (en_US) and Czech (cs_CZ).
- **Modern Workflow**: Integrated `@wordpress/scripts` for build automation.
- **Quality Control**: 
  - ESLint (JS)
  - Prettier (Format)
  - PHPCS (WordPress Coding Standards)
- **Documentation**: Professional `README.md`, `readme.txt`, and `SECURITY.md`.
- **Infrastructure**: 
  - `.distignore` for clean production ZIPs.
  - PHPUnit test suite with `phpunit.xml.dist` and bootstrap.
  - Initial `.pot` translation template.
- **Modular Architecture**: PSR-4-like structure with Composer for PHP dependency management.
