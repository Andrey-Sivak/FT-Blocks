== Description ==

FT Blocks is a collection of custom Gutenberg blocks designed specifically for photography websites. Built with modern development practices, it provides a seamless editing experience while maintaining high performance and code quality.

= Features =

* **Hero Block** - Create stunning hero sections with customizable content
* **Bilingual Support** - Full internationalization for English and Czech languages
* **Modern Development** - Built with React and WordPress block editor APIs
* **Performance Optimized** - Lightweight and fast-loading blocks
* **Accessibility Ready** - Following WCAG guidelines for inclusive design

= Supported Languages =

* English (en_US)
* Czech (cs_CZ)

= Developer Friendly =

* Clean, well-documented code
* Follows WordPress Coding Standards
* Integrated linting tools (ESLint, Prettier, PHPCS)
* Built with @wordpress/scripts for modern development workflow

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Plugins → Add New
3. Search for "FT Blocks"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Navigate to Plugins → Add New → Upload Plugin
3. Select the ZIP file and click "Install Now"
4. Activate the plugin after installation

= After Activation =

1. Create or edit a page/post
2. Click the "+" button to add a block
3. Search for "FT" to find FT Blocks
4. Select the block you want to use

== Frequently Asked Questions ==

= What WordPress version is required? =

FT Blocks requires WordPress 6.0 or higher to ensure compatibility with the latest block editor features.

= Does this plugin work with my theme? =

Yes! FT Blocks is designed to work with any properly coded WordPress theme that supports the block editor (Gutenberg).

= Can I customize the block styles? =

Yes, you can customize block styles through your theme's CSS or by using the WordPress Customizer. The blocks are built to respect your theme's design system.

= Is this plugin translation ready? =

Absolutely! FT Blocks is fully internationalized and includes support for English and Czech languages. Additional translations can be added using standard WordPress translation tools.

= How do I report bugs or request features? =

Please use the plugin's GitHub repository for bug reports and feature requests. You can also contact the author directly through the support forum.

= Is this plugin compatible with page builders? =

FT Blocks is designed for the WordPress block editor (Gutenberg). Compatibility with third-party page builders is not guaranteed.
